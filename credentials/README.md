# Thư mục credentials

Đặt file OAuth Google tại đây:

- `client_secret.json` — tải từ [Google Cloud Console](https://console.cloud.google.com/)
- `accounts.json` — tự tạo khi đăng nhập tài khoản YouTube
- `tokens/` — token OAuth (tự tạo)

Xem hướng dẫn chi tiết trong `README.md` ở thư mục gốc dự án.
